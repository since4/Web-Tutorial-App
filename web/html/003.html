What Can JavaScript Do?

JavaScript can change HTML attributes.

In this case JavaScript changes the src (source) attribute of an image.

        
        <!--Body content-->
        <button onclick="document.getElementById('myImage').src=
            './images/pic_bulbon.gif'">Turn on the light</button>
        <img id="myImage" src="./images/pic_bulboff.gif" style="width:100px">
        <button onclick="document.getElementById('myImage').src=
            './images/pic_bulboff.gif'">Turn off the light</button>
