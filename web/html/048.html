My First Google Map

Now Google wants an API key to authenticate users to access their APIs: https://developers.google.com/maps/documentation/javascript/get-api-key 

        
        <!--Body content-->
        <h1>My First Google Map</h1>

        <div id="map" style="width:100%;height:500px"></div>

        <script>
        function myMap() {
          var mapCanvas = document.getElementById("map");
          var mapOptions = {
            center: new google.maps.LatLng(51.508742,-0.120850),
            zoom: 5
          };
          var map = new google.maps.Map(mapCanvas, mapOptions);
        }
        </script>
        <p>
            Now Google wants an API key to authenticate users to access their APIs:
            https://developers.google.com/maps/documentation/javascript/get-api-key
        </p>
        <script 
            src="https://maps.googleapis.com/maps/api/js?callback=myMap">                   
        </script>
