What Can JavaScript Do?

JavaScript Statements

- Statements are separated by semicolons.

The variables x, y, and z are assigned the values 5, 6, and 11:

- Number can be written with or without decimals.

- Strings can be written with double or single quotes.

- JavaScript uses arithmetic operators to compute values (just like algebra).

- Expressions compute to values.

        
        <!--Body content-->
        <h2>JavaScript Statements</h2>
        <p>Statements are separated by semicolons.</p>
        <p>The variables x, y, and z are assigned the values 5, 6, and 11:</p>
        <p id="demo1"></p>
        <script>
            var x = 5;
            var y = 6;
            var z = x + y;
            document.getElementById("demo1").innerHTML = z;
        </script>
        <!--new example-->
        <p>Number can be written with or without decimals.</p>
        <p id="demo2"></p>
        <script>
            document.getElementById("demo2").innerHTML = 10.50;
        </script>
        <!--new example-->
        <p>Strings can be written with double or single quotes.</p>
        <p id="demo3"></p>
        <script>
            document.getElementById("demo3").innerHTML = 'John Doe';
        </script>
        <!--new example-->
        <p>JavaScript uses arithmetic operators to compute values 
            (just like algebra).</p>
        <p id="demo4"></p>
        <script>
            document.getElementById("demo4").innerHTML = (5 + 6) * 10;
        </script>
        <!--new example-->
        <p>Expressions compute to values.</p>
        <p id="demo5"></p>
        <script>
            document.getElementById("demo5").innerHTML = "John" + " "  + "Doe";
        </script>
        <p id="demo6"></p>
        <script>
            var x = 5;
            // var x = 6; I will not be executed
            document.getElementById("demo6").innerHTML = x;
