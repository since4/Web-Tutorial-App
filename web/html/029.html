What Can CSS Do?

Note: The height and width properties do not include padding, borders, or margins; they set the height/width of the area inside the padding, border, and margin of the element!

Set the height and width of an element

This div element has a height of 200px and a width of 500px:

Set the max-width of an element

This div element has a height of 200px and min-width of 400 and a max-width of 500px:

Resize the browser window to see the effect.

        
        <!--Internal Stylesheet-->
        <style>
            div.dims1 {
                height: 200px;
                width: 500px;
                background-color: powderblue;
            }
            div.dimsmax1 {
                max-width: 500px;
                min-width: 400px;
                height: 200px;
                background-color: powderblue;
            }
        </style>
        <!--Body content-->
        <p>
            <strong>Note:</strong> The <code class="w3-codespan">height</code> 
            and <code class="w3-codespan">width</code> 
            properties do not include padding, borders, 
            or margins; they set the height/width of the area 
            inside the padding, border, 
            and margin of the element!
        </p>               
        <div class="dims1">
            <h2>Set the height and width of an element</h2>
            <p>This div element has a height of 200px and a width of 500px:</p>
        </div>
        <div class="dimsmax1">
            <h2>Set the max-width of an element</h2>
            <p>This div element has a height of 200px and
                min-width of 400 and a max-width of 500px:</p>
            <p>Resize the browser window to see the effect.</p>
        </div>
